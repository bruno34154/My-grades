# My-grades

## Descrição:
Este projeto com o intuito de aprimorar os estudos de javascript, atravês de uma plataforma que calcula a nota media de boletins e mostra se o aluno foi aprovado ou não.

## Tecnologias usadas:

* HTML
* CSS
* Javascript
